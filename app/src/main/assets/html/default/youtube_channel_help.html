Using multiple Youtube channels

It is impossible for Watchlater to let you choose which of your channel's watchlater playlists to add to. If you don't like the channel being used by Watchlater or the selection seems random, you have the following possibilities:

  • Visit the advanced Youtube account settings page and select the desired channel as your default channel. It might take a while until the change takes effect.

  • Set up seperate logins for the desired Google+ accounts (channels).
